 1. Описание функций

Код содержит несколько функций для выполнения типовых задач:

 `factorial(n)`: Рассчитывает факториал числа $n$ ($n!$).
 `findMax(arr)`: Корректная функция для поиска максимального элемента в массиве.
 `findMaxFixed(arr)`: Исправленная версия функции для сравнения и проверки логики.
 `findMaxBroken(arr)`: Заранее "сломанная" функция с логической ошибкой, предназначенная для отладки.

 2. Модульное тестирование (Jest-Lite)

В приложении используется Jest-Lite (упрощенная версия Jest, работающая прямо в браузере) для автоматической проверки корректности функций.

 Назначение: Тесты проверяют `factorial()` и `findMaxFixed()`.
 Ключевой момент: Тест для `findMaxBroken()` намеренно проваливается (FAILED), когда ему передается массив с отрицательными числами (например, `[-5, -2, -9]`). Это указывает на проблему, которую необходимо решить.

 3. Задача на отладку (DevTools)

Цель этой части — использовать встроенные инструменты браузера для поиска и исправления ошибки в функции `findMaxBroken()`.
